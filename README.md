# Baseball Prototype

1台の端末を使って、青軍と赤軍で対戦する野球ゲームです。画面の上半分がピッチャー、下半分がバッターのエリアになり、投球、打撃、送球をタッチ操作で行います。

スマートフォンやタブレットの横持ちを想定しています。PCでもマウス操作で遊べます。

## 遊び方

1. `index.html`を開きます。
2. タイトル画面で試合形式を選びます。
   - `Playing (9回)`: 9回制
   - `Playing (3回)`: 3回制
3. `PLAY BALL!`の表示が消えたら試合開始です。

## 起動方法

Windowsでは、リポジトリ内の`run_static.bat`を実行するとブラウザが開きます。

```bat
run_static.bat
```

ポートを変えたい場合は、番号を指定できます。

```bat
run_static.bat 9000
```

Pythonが使える環境なら、次のコマンドでも起動できます。

```sh
python -m http.server 8765 --bind 127.0.0.1
```

そのあとブラウザで `http://localhost:8765/index.html` を開いてください。

## 画面の見方

- 上側: ピッチャー側の操作エリア
- 下側: バッター側の操作エリア
- 中央付近: 投球と打撃のプレイエリア
- `B / S / O`: ボール、ストライク、アウトのカウント
- `青軍` / `赤軍`: 各チームの得点
- `1回表`など: 現在のイニング

表は青軍の攻撃、裏は赤軍の攻撃です。

## 操作

### ピッチャー

ピッチャー側のエリアでボールを置き、相手側へスワイプして投げます。

スワイプの速さで球速が変わります。指の軌道を曲げると、投球にも曲がりがつきます。

### バッター

バッター側のエリアをタッチするとバットを構えます。ボールに合わせて、投手方向へ素早くスワイプするとスイングします。

タイミングとバットの当たり方で打球の強さや方向が変わります。強い打球は青く表示され、外野側の壁を越えるとホームランになります。

### 守備

打球が出るとランナーが走り、守備モードになります。

ボールが止まって赤くなったら、ボール付近をタッチして拾えます。拾ったボールを、光っている塁やホームへ向けて弾くように送球してください。ランナーより先に送球が到達するとアウトになります。

## 基本ルール

- 3ストライクで三振アウトです。
- 4ボールでフォアボールです。
- 3アウトでチェンジです。
- ヒットすると打者と塁上のランナーが自動で進塁します。
- ランナーがホームに到達すると得点です。
- 最終回終了時に得点が多いチームの勝ちです。
- 最終回裏で後攻の赤軍が勝ち越した場合は、その時点で試合終了です。
- 同点の場合は引き分けです。延長戦はありません。

## リセット

試合中に左上の`リセット`を3秒長押しすると、3回制または9回制を選んで最初からやり直せます。

試合終了後は`もう一度`で同じ試合形式を最初から遊べます。

## セーブ

試合中の状態はブラウザに自動保存されます。ページを再読み込みしても、途中から再開できます。

新しく試合を始めたとき、リセットしたとき、試合が終了したときは保存データが消えます。

## 音について

投球、打撃、アウト、得点、ホームランなどに効果音があります。ブラウザの仕様により、最初のタッチやクリック後に音が鳴り始めることがあります。
